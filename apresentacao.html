Pinax, o que é isso?

Uma plataforma para desenvolvimento rápido de websites.

Por padrão o Pinax trás uma série de templates prontos de projetos web, com funcionalidades muito comuns, como as citadas anteriormente e muitas outras.

$ pip install Pinax
$ pinax-admin setup_project -b basic minha_redesocial
$ python manage.py syncdb
$ python manage.py runserver
