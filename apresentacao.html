Funcionalidades que devem ser desenvolvidas e que toda rede social tem:

cadastro de usuário login/logoff do usuario validação de e-mail modificar senha esqueci a senha configurações de recebimento de e-mail enviar recado pra outro usuário adicionar como amigo notificar conteúdo indevido foto do perfil perfil do usuario pesquisa de usuário exclusão de conta de usuário comunidades com lista de discussão pesquisa de grupos e no forúm microblogging

Uma plataforma para desenvolvimento rápido de websites.

Por padrão o Pinax trás uma série de templates prontos de projetos web sobre o framework Django. Agrega funcionalidades muito comuns, como as citadas anteriormente e muitas outras.

Você obtém um projeto Django totalmente configurado e pronto para ser usado.

Tipos de projetos prontos do Pinax

Os mais interessantes para a criação de redes sociais são:

  • private_beta
    • Este projeto demonstra o uso de uma lista de espera e de códigos de acesso para sites em beta privado.
  • social
    • Este projeto demonstra um site de rede social com profiles, possibilidade de adicionar como amigo, fotos, grupos, wikis, favoritos, envio de mensagens entre usuários (scrap) etc.

Existem outros projetos de CMS, Intranet, etc.

É complicado instalar e ter minha primeira rede social funcionando?

Tire suas próprias conclusões:

$ pip install Pinax
$ pinax-admin setup_project -b private_beta minha_redesocial
$ python manage.py syncdb
$ python manage.py runserver

Possibilidades, e agora?

    • Com o pinax você obtém o básico que seu projeto precisa para começar a ser desenvolvido.
    • Assim, você pode manter o foco no que é interessante e no diferencial que a sua rede social vai ter em relação as existentes.

Mantenha o foco no diferencial

    • Redes sociais que não tiveram um foco específico como diferencial, acabaram indo por água a baixo.
      • Gazzag.com - Clone das funcionalidades do Orkut e fechou as portas.
      • 1grau.com.br - Também era uma cópia das funcionalidades do Orkut e não existe mais.

"Como vou fazer diferente? Tem tanta rede social por ai..."

Faça diferente

    • Descubra um nicho em aberto, um perfil de um grande grupo de pessoas que foi inexplorado ou mal explorado.
      • LinkedIn.com - Rede social de negócios.
      • Last.FM - Rede social de música.
      • Livemocha.com - Rede social de pessoas que querem aprender outras linguas.

"Isso só acontece no exterior? E aqui no Brasil, dá pra inovar?"

Redes sociais genuinamente brasileiras

    • No Brasil também tem gente fazendo redes socias de nicho com qualidade.
      • Skoob.com.br - Rede social de leitores.
      • Filmow.com - Rede social de cinéfolos.
      • AtePassar.com - rede social de estudantes de concursos públicos.

Desenvolvimento da rede social AtéPassar

    • Começamos com o Pinax.
    • Evoluimos a rede adicionando funcionalidades diferênciadas:
      • Plano de estudo com acompanhamento dos estudos;
      • Resolução de questões;
      • Estátisticas de questões resolvidas;
      • E muitos outros vindo por ai...
